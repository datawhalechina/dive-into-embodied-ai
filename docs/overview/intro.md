# Dive into Embodied AI — 动手学具身智能

面向求职与转行的具身智能开源实践教程，用基础知识打底、项目实践贯穿、求职能力收口，帮助学习者从零构建具身智能项目，积累真实工程经验。

## 本教程适合谁

- 应届生 / 在读研究生，想进入具身智能方向
- 互联网算法工程师（CV / NLP / 推荐），想转行具身智能
- 传统机器人工程师，想补齐 VLM / VLA / 模仿学习等新技术
- 自动驾驶 / CV 背景，想切入机器人赛道

## 如何使用本教程

1. **先看总览**：通过[技术路线全景图](./embodied-ai-roadmap)建立全局认知
2. **选择路径**：根据你的背景在[学习路径推荐](./learning-path)中找到适合的路线
3. **基础篇打底**：按路线推荐顺序学习基础模块
4. **实践篇动手**：选择感兴趣的本体方向完成项目
5. **求职篇收口**：用项目经验对接岗位要求，准备面试
