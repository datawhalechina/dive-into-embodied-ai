# 学习路径推荐

:::info 占位页面
本页将根据不同背景推荐 4 条学习路线，每条给出推荐阅读顺序。
:::

## 路线 A：学生 / 应届生

重点：先补基础 → 做 1 个机械臂项目 → 做 1 个 VLA 小项目 → 整理简历

## 路线 B：互联网算法转行

重点：利用 ML/DL/RL 基础 → 补机器人学、ROS、仿真 → 做数据采集 + VLA 项目

## 路线 C：传统机器人转具身智能

重点：利用控制、运动学、ROS 基础 → 补 VLM/VLA、模仿学习 → 做语言驱动机械臂项目

## 路线 D：自动驾驶 / CV 背景转入

重点：利用感知、BEV、多模态经验 → 补机器人控制和 action 建模 → 做移动机器人 / VLA 项目
