# 基础篇：建立具身智能工程底座

基础篇围绕后续项目实践需要什么就补什么，让学习者具备看懂代码、跑通环境、理解论文、进入项目实践的基础能力。

## 模块一览

| 模块 | 核心问题 | 服务于 |
|------|---------|--------|
| [具身智能入门](./embodied-ai-intro/placeholder) | 这个方向到底在做什么？ | 全局认知 |
| [机器人基础](./robotics-basics/placeholder) | 为什么需要运动学、控制、坐标系？ | 机械臂、四足项目 |
| [ROS2 工程基础](./ros2-basics/placeholder) | 机器人系统怎么组织起来？ | 所有实践项目 |
| [仿真工具基础](./simulation/placeholder) | 怎么在仿真中跑机器人？ | 所有实践项目 |
| [强化学习与控制](./rl-for-robotics/placeholder) | RL 怎么用在机器人上？ | 四足、双足项目 |
| [数据采集与模仿学习](./imitation-learning/placeholder) | 数据到底长什么样？ | 机械臂、VLA 项目 |
| [VLM / VLA / World Model](./vlm-vla-world-model/placeholder) | 当前具身大模型在做什么？ | VLA 相关项目 |
